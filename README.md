# 🎣 Phishing Email Simulator

A cybersecurity awareness project built using HTML, CSS, and JavaScript that demonstrates the difference between phishing emails and legitimate emails.

This project helps users identify common phishing techniques such as:

* Spoofed sender domains
* Suspicious links
* Urgent/threatening language
* Generic greetings
* Header mismatches

---

## 🚀 Features

* Interactive phishing email simulation
* Safe vs unsafe email comparison
* Suspicious element highlighting
* Inspector panel with explanations
* Raw email header analysis
* Responsive cybersecurity-themed UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
